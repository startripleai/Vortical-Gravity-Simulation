# Vortical Gravity Simulation Engine
This repository provides the numerical verification and visualization tools for the **Vortical Gravity** theory (Absolute Quadrilogy).

## Key Features
- **Dark Matter Resolution**: Simulates galactic rotation curves without non-baryonic particles.
- **G-Recovery Analysis**: Visualizes how gravitational coupling restores to $G_{max} \approx 5.46 G_0$ in low-density regimes.
- **$10^{122}$ Solution**: Includes the logic for resolving the Cosmological Constant Problem.

## Quick Start
1. Clone this repo: `git clone https://github.com`
2. Install requirements: `pip install -r requirements.txt`
3. Run the notebook: `main_visualization.ipynb`
